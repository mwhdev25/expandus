## About Figma

Figma is the industry-standard collaborative design tool for UI/UX teams. It runs entirely in the browser, enabling real-time collaboration across designers, product managers, and developers. With the addition of FigJam (whiteboarding), Dev Mode (developer handoff), and AI features, Figma is more than a design tool — it's the creative hub for product teams.

**Website:** [figma.com](https://figma.com)

## 🎯 Use Cases

- **UI/UX Design:** Wireframes, high-fidelity mockups, design systems
- **Product Prototyping:** Interactive click-through demos for user testing
- **Design Systems:** Shared component libraries for product consistency
- **Developer Handoff:** Annotated specs and auto-generated CSS/code
- **Presentations:** Visual decks and pitch presentations
- **Diagramming:** Flowcharts, user journeys, and system architecture (FigJam)

---

## ✅ Pros

- Industry standard — expected skill for all UI/UX designers
- Browser-based — no installation, works on any OS
- Real-time multiplayer collaboration is category-defining
- Massive plugin ecosystem (2,000+ plugins)
- Dev Mode bridges design-development gap effectively

---

## ⚠️ Limitations

- Requires internet — limited offline functionality
- Can get slow with complex files and many components
- Free plan limited to 3 projects and 2 editors
- Advanced features require costly Organization tier
